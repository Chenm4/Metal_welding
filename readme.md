启动前需要有一个数据库，数据库的配置文件在config.py中（更改一下mysql的密码和用户为师兄的）,默认通过backend\scripts\init_db.py这个脚本创建数据库，生成的数据库中默认有两个一个是admin用户一个是viewer用户，密码都是123456。师兄可以登录admin用户然后选择导入data\batch_1\625试验数据_修改过后.xlsx这个xlsx来添加数据内容。
目前启动命令是：
cd frontend
npm run dev
然后新开一个窗口执行：uvicorn backend.main:app --reload --host 0.0.0.0 --port 8004
然后打开http://localhost:5173/ 即可
