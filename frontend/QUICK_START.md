# 🚀 快速启动指南

## 前提条件

1. 确保后端服务已启动在 `http://localhost:8004`
2. Node.js 版本需要 >= 22.12 或 >= 20.19

## 启动步骤

### 1. 进入前端目录
```bash
cd E:\value_code\Metal_welding\my-react-app
```

### 2. 安装依赖（如果还没安装）
```bash
npm install
```

### 3. 启动开发服务器
```bash
npm run dev
```

### 4. 打开浏览器访问
```
http://localhost:5173
```

## 🎯 使用流程

### 1. 登录系统
1. 打开浏览器访问 http://localhost:5173
2. 输入用户名和密码
3. 点击"登录"按钮

### 2. 查看数据
1. 登录后自动跳转到数据管理页面
2. 侧边栏显示所有可用的数据集
3. 点击数据集名称切换

### 3. 搜索数据
1. 在搜索框输入关键词
2. 按回车或点击搜索按钮
3. 系统会在可搜索字段中查找

### 4. 筛选字段（按分类）
1. 勾选/取消勾选分类复选框
2. 物性参数（蓝色）
3. 工艺参数（绿色）
4. 状态参数（橙色）
5. 性能指标（红色）

### 5. 新增数据（管理员）
1. 点击"新增数据"按钮
2. 在弹窗中填写各字段
3. 必填字段会标记为红色
4. 点击"保存"提交

### 6. 编辑数据（管理员）
1. 点击数据行的"编辑"按钮
2. 修改字段值
3. 点击"保存"更新

### 7. 删除数据（管理员）
- **单个删除**: 点击"删除"按钮
- **批量删除**: 
  1. 勾选多行数据
  2. 点击"批量删除"按钮
  3. 确认操作

### 8. 导入文件（管理员）
1. 点击侧边栏底部的"导入 CSV 文件"按钮
2. 拖拽或点击上传文件
3. 选择导入模式：
   - **新建数据集**: 输入新数据集名称（如 batch_5）
   - **追加数据**: 选择现有数据集
4. 点击"确认导入"
5. 等待导入完成，查看结果统计

## 🐛 常见问题

### 1. 登录后立即退出
**原因**: Token 验证失败或后端未启动
**解决**: 
- 检查后端是否运行在 http://localhost:8004
- 检查浏览器控制台错误信息

### 2. 数据列表为空
**原因**: 数据库中没有数据
**解决**: 
- 使用管理员账号导入 CSV 文件
- 或者使用后端脚本初始化数据

### 3. 导入文件失败
**原因**: 文件格式不正确或列名不匹配
**解决**: 
- 确保文件是 CSV 或 Excel 格式
- 检查列名是否符合规范（物性_xxx, 工艺_xxx 等）
- 查看错误提示信息

### 4. 页面显示异常
**原因**: 浏览器缓存或版本不兼容
**解决**: 
- 清除浏览器缓存
- 使用 Chrome/Edge 最新版本
- 按 Ctrl+Shift+R 强制刷新

## 💡 小技巧

1. **键盘快捷键**: 
   - 搜索框按回车直接搜索
   - ESC 关闭弹窗

2. **数据完整度**: 
   - 顶部进度条显示当前数据集的完整度
   - 绿色表示达标（>90%）
   - 红色表示需要补充数据

3. **表格操作**:
   - 点击列标题可以排序
   - 表格支持横向滚动
   - 可以调整每页显示数量

4. **批量操作**:
   - 表格顶部复选框可以全选
   - 选中后显示已选数量
   - 批量删除需要二次确认
